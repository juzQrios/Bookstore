## Get Started

### Setup

1. Clone this repository.

```bash
git clone git@github.com:juzQrios/bookstore.git
cd bookstore
```

2. Install dependencies.

```bash
npm install
```

3. Start the local server.

```bash
npm start
```

4. Open <http://localhost:3000/> in a browser.
